## TabBatch PDF v2.0.0 🎉

A complete, production-grade rewrite that turns a window full of tabs into one clean, bookmarked PDF — fast, private, and fully offline.

### Highlights
- 🗂 **Tab picker** — choose exactly which tabs to capture (this window or all windows), with favicons and select-all/none.
- 📄 **True vector capture** via `Page.printToPDF` — selectable text, clickable links, tiny files.
- 📐 **Two modes** — Continuous (one tall page) or Paged (A4 / Letter / Legal / A3 / A5 / Tabloid, portrait/landscape).
- 🔖 **Merge enhancements** — PDF bookmarks, page numbers, a clickable table of contents, and document metadata.
- 🖼 **Manager** — search, multi-select, drag-to-reorder, inline rename, preview, per-file download, and recapture.
- ⚙️ **Settings page** (synced), 🌓 light/dark/system themes, ⌨️ keyboard shortcut, 🖱 right-click menus.
- 🧠 **Reliability** — cancellable batches, per-tab timeouts/retries, MV3 keep-alive, oversize fallback, and "one bad tab won't sink the run".
- 🔒 **100% local** — no servers, no tracking; pdf-lib is bundled (no CDN).

### Quality
- ✅ 53 automated tests (utilities, settings model, IndexedDB layer)
- ✅ Real anti-aliased PNG icons at all required sizes
- ✅ Full docs: README, PRIVACY, CHANGELOG, and Chrome Web Store listing copy

### Install (developer mode)
1. Download `tabbatch-pdf.zip` below and unzip it (or clone the repo).
2. Open `chrome://extensions/`, enable **Developer mode**.
3. **Load unpacked** → select the folder containing `manifest.json`.

See the [README](https://github.com/shahriar-ahmed-seam/tabbatch-pdf#readme) for full usage and the [CHANGELOG](https://github.com/shahriar-ahmed-seam/tabbatch-pdf/blob/main/CHANGELOG.md) for details.
